```
1、特殊字符
代表包含特殊意义的字符，若是想表达自身的含义，则需要在前面加转义字符\

^开头，$结尾

集合 []

?[0,1],+[1,多次],*[任意次]

重复次数限定{n},{n,m},{n,}

非打印字符，即无法打印出来的字符
换页\f,换行\n，回车\r，制表符\t，垂直制表符\v，空白符\s,等效于[\f\n\r\t\v]

```

示例

```js
let pattern = /^.{1,10}$/;
let pattern = /^[0-9a-zA-Z_\-]{1,10}$/;
let pattern = /^\S{1,10}$/;
let pattern = /^[\u4e00-\u9fa5]{1,10}$/;
let pattern = /(^[0-9]{15}$)|(^[0-9]{17}[\dXx]{1}$)/;
let pattern = /^[0-9]{0,6}(\.[0-9]{1,4})?$/;
let pattern = /^[a-zA-Z0-9_\-]+@[a-zA-Z0-9_\-]+(\.[a-zA-Z0-9_\-]+)+$/;
```
